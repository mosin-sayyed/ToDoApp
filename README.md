# Flask To-Do App

A simple and efficient To-Do List web application built using Flask and SQLite.  
This project helps users manage daily tasks easily.

---

## Features

- Add new tasks
- Update tasks
- Delete tasks
- Store data in SQLite
- Clean and user-friendly interface

---

## Tech Stack

- Python (Flask)
- SQLite (SQLAlchemy)
- HTML, CSS
- Git & GitHub
